How to Play

Objective: Cover manholes to save pedestrians from falling!

Controls:

• Arrow Keys ← → to select manhole

• Space or COVER button to place cover

Scoring:

• +10 points for saving pedestrians

• +5 points when pedestrians cross safely

• 3 misses = Game Over

MANHOLE
Arrow Keys: Move | Space: Cover | Tap buttons for mobile
